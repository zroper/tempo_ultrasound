EXAMPLE Directory.

This directory contains sample programs (NODES) and test LC files for them.


    EXAMPLE0.C - NODE that sends a message to ELSIE
    EXAMPLE1.C - Illustrate tgcbLogf() and tgcbMessagef() callbacks
    EXAMPLE2.C - NODE that receives a message from another NODE
    EXAMPLE3.C - NODES that make queries of other NODES
    EXAMPLE4.C - Simple example of LC control program
    
    EXAMPLET.C - Examples of how to communicate with TEMPO or COBALT
